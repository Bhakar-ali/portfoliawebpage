You might have been awestruck quite a few times while scrolling beautiful websites with complicated yet smooth features. All of those visible site features are created using front-end development (also known as “front-end web development”), and the people who create them are known as front-end developers. While web design is concerned with how a website appears, front-end development is concerned with how that design is deployed on the web.
